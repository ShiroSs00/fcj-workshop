### Mục tiêu Tuần 11:

- Tìm hiểu các tính năng nâng cao của ASP.NET Core: logging, caching, và tối ưu hóa hiệu suất.
- Thực hành xây dựng các APIs có thể mở rộng với logging và monitoring phù hợp.
- Tìm hiểu về tài liệu API với Swagger/OpenAPI.

### Thành tựu Tuần 11:

- Thành công tìm hiểu và triển khai structured logging với Serilog trong ASP.NET Core.

- Cấu hình ghi logs tới nhiều sinks: Console, File, và AWS CloudWatch.

- Triển khai in-memory và distributed caching strategies để cải thiện hiệu suất API.

- Thêm cache invalidation và TTL policies để đảm bảo tính nhất quán của dữ liệu.

- Thành công tích hợp tài liệu Swagger/OpenAPI vào API.

- Tài liệu hóa tất cả các API endpoints với XML comments và examples.

- Kiểm thử chức năng API qua interactive Swagger UI.

- Tích hợp comprehensive monitoring với CloudWatch để phân tích logs real-time.

- Đạt được khả năng xây dựng production-ready, well-documented, và monitored APIs.
