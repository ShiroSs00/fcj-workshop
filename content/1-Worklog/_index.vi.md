## Tổng quan Thực tập

Nhật ký công việc này ghi chép toàn bộ chương trình thực tập First Cloud Journey 12 tuần. Chương trình bao gồm các dịch vụ AWS cloud, kiến trúc serverless, và phát triển full-stack với ASP.NET Core.

**Thời gian:** 8 tháng 9 - 29 tháng 11, 2025 (12 tuần)

**Trọng tâm chương trình:** AWS Cloud Services + Phát triển ASP.NET Core API

**Tuần 1 (08/09 - 12/09):** [Làm quen với AWS và các dịch vụ cơ bản](1.1-week1/)

- Nền tảng AWS và các danh mục dịch vụ
- Thiết lập AWS Console và AWS CLI
- Thực hành EC2

**Tuần 2 (15/09 - 20/09):** [Tìm hiểu AWS IAM và VPC](1.2-week2/)

- Identity and Access Management (IAM)
- Virtual Private Cloud (VPC) và mạng
- Security groups và cấu hình mạng

**Tuần 3 (22/09 - 27/09):** [Bài thực hành S3 và DynamoDB](1.3-week3/)

- Simple Storage Service (S3) để lưu trữ objects
- DynamoDB cơ sở dữ liệu NoSQL
- Quản lý lưu trữ và dữ liệu

**Tuần 4 (29/09 - 04/10):** [Lambda, API Gateway, và CloudWatch](1.4-week4/)

- Serverless computing với Lambda
- Tạo REST API với API Gateway
- Giám sát với CloudWatch

**Tuần 5 (06/10 - 11/10):** [Bài thực hành cho các dịch vụ trước đó](1.5-week5/)

- Thực hành hands-on với IAM, VPC, S3, và DynamoDB
- Xây dựng các dự án tích hợp
- Khắc phục sự cố và tối ưu hóa

**Tuần 6 (13/10 - 18/10):** [Thực hành Lambda, API Gateway, và CloudWatch](1.6-week6/)

- Phát triển ứng dụng serverless
- Tích hợp và kiểm thử REST API
- Giám sát và ghi nhật ký ứng dụng

**Tuần 7 (20/10 - 25/10):** [CloudFormation và tự động hóa cơ sở hạ tầng](1.7-week7/)

- Infrastructure as Code (IaC) với CloudFormation
- EC2 Auto Scaling và Load Balancing
- Triển khai ứng dụng đa tầng

**Tuần 8 (27/10 - 01/11):** [CI/CD pipelines và DevOps](1.8-week8/)

- CodePipeline, CodeBuild, và CodeDeploy
- Tự động hóa deployment pipelines
- Distributed tracing với X-Ray

**Tuần 9 (03/11 - 08/11):** [Nền tảng ASP.NET Core](1.9-week9/)

- Cấu trúc project ASP.NET Core
- Dependency Injection và middleware
- Entity Framework Core
- Phát triển REST API

**Tuần 10 (10/11 - 15/11):** [ASP.NET Core với tích hợp AWS](1.10-week10/)

- JWT authentication và authorization
- Tích hợp Cognito để quản lý users
- Tích hợp S3 và DynamoDB
- Phát triển API an toàn

**Tuần 11 (17/11 - 22/11):** [Các tính năng nâng cao của ASP.NET Core](1.11-week11/)

- Structured logging với Serilog
- Chiến lược caching (in-memory và distributed)
- Tài liệu Swagger/OpenAPI
- Tích hợp CloudWatch logging

**Tuần 12 (24/11 - 29/11):** [Dự án nền tảng chia sẻ video](1.12-week12/)

- Thiết kế và kiến trúc ứng dụng hoàn chỉnh
- Triển khai full-stack với dịch vụ AWS
- Triển khai và tự động hóa CI/CD
- Giám sát và tối ưu hóa production

## Kết quả Chương trình

Bằng cách hoàn thành chương trình thực tập 12 tuần này, những người tham gia sẽ đạt được:

- Hiểu biết toàn diện về dịch vụ AWS cloud và kiến trúc
- Kinh nghiệm hands-on với Infrastructure as Code và các thực hành DevOps
- Kỹ năng phát triển full-stack API sử dụng ASP.NET Core
- Khả năng thiết kế và triển khai các ứng dụng cloud có thể mở rộng, an toàn
- Kinh nghiệm xây dựng các dự án thực tế tích hợp nhiều dịch vụ AWS
- Kiến thức về giám sát, ghi nhật ký, và operational excellence

## Công nghệ và Dịch vụ Được Bao gồm

**Dịch vụ AWS:**

- Compute: EC2, Lambda
- Storage: S3, EBS
- Database: DynamoDB, RDS
- Networking: VPC, API Gateway, CloudFront
- Security: IAM, Cognito
- DevOps: CloudFormation, CodePipeline, CodeBuild, CodeDeploy
- Monitoring: CloudWatch, X-Ray

**Development Stack:**

- ASP.NET Core framework
- Entity Framework Core
- JWT authentication
- Serilog logging
- Swagger/OpenAPI documentation
